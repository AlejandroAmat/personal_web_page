Projects

(i) Skin Tone Detector

Image Processing Project focused on detecting which pixels of an Input image are skin. The main idea of the algorythm is to create a Normalized Histogram of skin pixel values obtained from all the Training Set Images (As we previously know the pixels with skin) and another one with the non-skin pixel values. When normalizing both histograms by their p(x_i) {x1=skin, x2=non-skin} we obtain two histograms that represent the conditioned probabilities of each succes. In order to compute if an input pixel is skin or not, we compare the probability of that level of colour (Actually Blue and Red Difference Crominance) through Bayesian Decision.
